# Modern Debugging in PHP - 2015
## Presented to Kansas City's PHP User Group
### April 1, 2015

####Slide system uses:
* npm
* php 5.4+
* composer
* reveal.js
* lmtm/prez
* slim (php)

####To Start Presentation:
````bash
cd php-example/public
php -S 127.0.0.1:8888
````
````bash
npm install -g prez
prez --server --watch
````
